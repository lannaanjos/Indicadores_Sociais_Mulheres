# Predição de Indicadores Sociais

Neste projeto, fazemos uso de um MLP para fazer a predição de indicadores socioeconômicos de mulheres nas capitais brasileiras no ano 2000.
Usamos dados vindos do IBGE, dos anos 1991 e 2000 para realizar essa predição.
O objetivo é verificar se o MLP consegue entender a correlação das tabelas 2, 4, 6, 8 e 19 e fazer "predições" razoáveis com base no aprendizado.
